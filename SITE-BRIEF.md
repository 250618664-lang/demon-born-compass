# Demon Born fast-test site brief

## Project type

Fast-test Roblox guide site for the rising `Demon Born Roblox` trend.

Working name: **Demon Born Compass**  
Domain: not purchased yet. Use Vercel preview or temporary deployment first.

## Current decision

`RESEARCH -> BUILD_READY_FOR_PREVIEW`

Build is allowed only as a conservative preview site using the evidence pack in `research/demon-born/`.

## Evidence base

Primary evidence:

- Roblox official game page/API for identity, creator, description, genre, controls and volatile metrics.
- Roblox group/votes/badges APIs for official metadata.
- User-supplied Google Trends and SERP screenshots for opportunity validation.

Third-party code/guide pages are **leads only** and cannot support game facts or code lists.

## Positioning

The site should target:

- Demon Born wiki
- Demon Born trello
- Demon Born codes
- Demon Born controls
- Demon Born beginner guide

It should not pretend to be official, not copy code lists, and not claim tested mechanics.

## Launch style

Static Astro site. No database, no ads, no analytics, no third-party images.

## Separation from Echoes Compass

This is the second site and must not be confused with Echoes Compass / Echoes of Aincrad.

- Use independent brand copy, page titles, metadata, source IDs and claims.
- Do not copy Echoes page text, navigation labels, footer wording or game facts.
- Do not reuse Echoes visual identity as-is; only reuse the general engineering pattern if helpful.
- Keep all Demon Born evidence under `research/demon-born/`.
- Keep all Demon Born site files under `sites/demon-born/`.

The first screen must show:

- unofficial status
- last checked date
- whether official Trello/wiki/codes are verified
- links to the Roblox game page and sources

## Hard boundaries

- No copied codes from Pocket Tactics, Destructoid, Beebom, Radio Times or other sites.
- No official Trello/Discord claim unless verified from official source or user-provided official channel.
- No clans/classes/tier-list/best-build claims without official or tested evidence.
- No Roblox thumbnails/media images unless rights are confirmed.
- Volatile metrics must show retrieval time or stay out of evergreen text.

## Success check

D3: Google discovers the site or pages.  
D7: any GSC impression/click for `Demon Born wiki`, `Demon Born trello`, `Demon Born codes`, or related variants.  
D14: decide Scale / Maintain / Freeze.
