# Demon Born Compass

**Unofficial Demon Born Roblox guide site.**

## What this is

- Unofficial fan guide for [Demon Born](https://www.roblox.com/games/79833666916700/Demon-Born) on Roblox.
- No affiliation with Roblox, SoraExp, or the Demon Born creators.
- Ads are disabled.
- No third-party code lists are copied or published.
- Evidence-backed status pages only — all facts trace to registered source IDs.

## What this is NOT

- No tier lists, best builds, or best clans.
- No breathing styles or blood demon arts content (not yet confirmed from official sources).
- No copied codes from Pocket Tactics, Destructoid, Beebom, Radio Times, or any other third-party site.
- No claim that an official Trello or Discord exists unless verified from an official source.

## Tech stack

- Astro + TypeScript, static output
- @astrojs/sitemap
- No database, no analytics, no ads

## Development

```bash
npm install
npm run dev      # local preview
npm run build    # production build + validation
npm run check    # TypeScript checks
npm run validate # content validation only
```
